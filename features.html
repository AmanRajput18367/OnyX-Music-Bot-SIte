🎵 Music Bot Commands

  • ▶️ play <query> - Play a song or playlist
  • ⏸️ pause - Pause the current track
  • ▶️ resume - Resume the current track
  • ⏭️ skip - Skip the current track
  • ⏹️ stop - Stop playback and clear queue
  • 📃 queue - Show the current queue
  • 🎧 nowplaying - Show current track info
  • 🔊 volume <0-100> - Adjust player volume
  • 🔀 shuffle - Shuffle the current queue
  • 🔁 loop - Toggle queue loop mode
  • 🗑️ remove <position> - Remove a track from queue
  • 🧹 clear - Clear the current queue
  • 📊 status - Show player status
  • ❓ help - Show this help message
